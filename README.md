# rugby-feed

Public JSON feed of rugby fixtures + UK broadcast info, consumed by a personal
Android app (source: [`lbrado/Personal`](https://github.com/lbrado/Personal),
`Rugby App/`). Published so the app can fetch updates over plain HTTPS without
needing credentials for the private source repo.

`fixtures.json` is generated by `Rugby App/scraper/espn_scraper.py` and
(once the publish step in `.github/workflows/rugby-fixtures.yml` is enabled)
overwritten here automatically 3×/day. No personal data — just fixture
kick-offs, teams, venues, and TV/streaming channels, all sourced from ESPN's
public API and a hand-maintained broadcaster map.
